# FastAPI Monitoring & Observability

This project demonstrates how to build a production-ready FastAPI application with a complete observability stack:

## 🚀 Features

- 📊 Prometheus Metrics
- 🔎 Distributed Tracing (OpenTelemetry)
- 🧵 Trace ↔ Log Correlation
- 📦 Dockerized Observability Stack
- 📈 Grafana Dashboards
- 🧠 Custom Metrics & Instrumentation
- 🏷️ Low Cardinality Best Practices

## 🏗 Architecture

FastAPI → OpenTelemetry → OTEL Collector → Prometheus / Tempo → Grafana
